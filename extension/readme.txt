== Installation ==
1. Primary: == Installation ==
	- In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
	- Type in Extension in the search form and press the 'Enter' key on your keyboard.
	- Click on the 'Activate' button to use your new theme right away.
	- Go to https://themefreesia.com/theme-instruction/extension/ for a guide on how to customize this theme.
	- Navigate to Appearance > Customize in your admin panel and customize to taste.

2. Alternate: Unzip the template file (extension.zip) that you have downloaded. Via FTP, upload the whole folder (extension) to your server and place it inside the http://yourdomain.com/wp-content/themes/folder(extension). Do not change directory name. The template files should thus be here now: http://yourdomain.com/wp-content/themes/extension/index.php (for example).
	- Log into your WP admin panel and click on the Appearance -> Theme tab. Now click on the Extension theme to activate it.
	- Want to enjoy Extension feature. Go to  Dashboard -> Appearance -> Customize and play with Extension Options
	- The theme is offered free of charge. If someone asked money for it, someone just tricked you.
